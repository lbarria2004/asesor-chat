# 🚀 Guía de Instalación - Asesor Previsional Virtual v1.1

## ✅ Ahora con Google Gemini AI

El chatbot ahora usa **Google Gemini AI** para respuestas inteligentes y naturales.

---

## 📋 Paso 1: Obtener API Key de Google Gemini (GRATIS)

1. Ve a **[Google AI Studio](https://aistudio.google.com/apikey)**
2. Inicia sesión con tu cuenta de Google
3. Clic en **"Create API Key"**
4. **Copia la API Key** generada

---

## 📋 Paso 2: Desplegar en Vercel

### Opción A: Desde GitHub (Recomendado)

1. Ve a **[vercel.com](https://vercel.com)** y crea cuenta
2. Clic en **"Add New..."** > **"Project"**
3. Importa: `lbarria2004/asesor-chat`
4. En **Environment Variables**, agrega:

| Name | Value |
|------|-------|
| `GEMINI_API_KEY` | `TU_API_KEY_DE_GOOGLE` |

5. Clic en **"Deploy"**
6. Espera 2-3 minutos
7. Copia tu URL (ej: `https://asesor-chat.vercel.app`)

---

## 📋 Paso 3: Instalar Plugin WordPress

1. Ve a **WordPress Admin** > **Plugins** > **Añadir nuevo**
2. Clic en **"Subir plugin"**
3. Selecciona `asesor-previsional-v1.1.0.zip`
4. **Instala y activa**

---

## 📋 Paso 4: Configurar Plugin

Ve a **WordPress Admin** > **Asesor Previsional** > **Configuración**:

| Campo | Valor |
|-------|-------|
| URL del Chat API | `https://tu-app.vercel.app` |
| Email de notificaciones | tu@email.com |

---

## ✅ Verificar

1. Abre tu sitio web
2. Verás el **botón flotante** naranja
3. Haz clic y conversa con el asesor
